## 函数

### abs

```go
func abs(a int) int {
   if a < 0 {
      return -a
   }
   return a
}
```

### min

```go
func min(a, b int) int {
   if a > b {
      return b
   }
   return a
}
```

### max

```go
func max(a, b int) int {
   if a > b {
      return a
   }
   return b
}
```
